# 图书借阅管理系统

## 项目简介
基于Java语言，SSM框架的图书借阅管理系统。分用户登录、密码修改，图书管理、图书借阅、图书归还、新增图书等功能。对六张数据库表的增删改查。
适合大作业、课程设计、毕业设计。

## 技术介绍
Java;Spring+Mybatis+SpringMVC;Mysql;JSP

## 开发工具
1.IntelliJ IDEA或者Myeclipse 3.Navicat 3.Mysql 4.JDK8

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
